# Brand Map — موقع تفاعلي رقمي لبناء العلامة التجارية

## 1. وصف المشروع
**Brand Map** هو أداة استشارية رقمية تفاعلية تُمكّن الشركات والوكالات التسويقية والرياديين من بناء خريطة علامتهم التجارية بشكل تفاعلي عبر 7 مراحل استراتيجية (السوق → الجمهور → القيمة → الشخصية → الرسالة → الهوية → الرؤية). يحلّ الموقع محلّ النسخة المطبوعة الفاخرة ويوفّر تجربة رقمية متكاملة مع حفظ التقدم ومشاركة الفرق.

**الجمهور المستهدف:** شركات، وكالات تسويق، رياديين، مديري علامات تجارية

**القيمة الأساسية:** بناء خريطة براند كاملة في 45 دقيقة عبر تجربة تفاعلية ذكية مع كشف تعارضات استراتيجية تلقائياً.

## 2. هيكل الصفحات
- `/` — الصفحة الرئيسية (تعريف بالمنتج + دعوة للتجربة)
- `/login` — تسجيل الدخول
- `/register` — إنشاء حساب جديد
- `/dashboard` — لوحة تحكم المستخدم (قائمة Brand Maps الخاصة به)
- `/brand-map/new` — إنشاء Brand Map جديد (الاختيار بين فردي/فريق)
- `/brand-map/:id` — صفحة اللعب التفاعلي (الـ 7 مراحل مع البطاقات)
- `/brand-map/:id/output` — ورقة Output Map النهائية (خريطة البراند الجاهزة)
- `/pricing` — صفحة الأسعار والخطط
- `/about` — عن Brand Map

## 5. التكامل الخلفي
- **Supabase Auth:** تسجيل دخول وإنشاء حساب
- **Supabase Database:** حفظ Brand Maps والإجابات والتعارضات
- **PDF Export:** تصدير Output Map كـ PDF (jsPDF + html2canvas)

## 6. خطة مراحل التطوير

### المرحلة 1: الصفحة الرئيسية + المصادقة + لوحة التحكم
- الهدف: إظهار واجهة الموقع الأساسية وتمكين المستخدم من التسجيل وإدارة مشاريعه
- المخرجات: الصفحة الرئيسية، صفحات تسجيل الدخول/الإنشاء، لوحة التحكم، قاعدة بيانات Supabase

### المرحلة 2: واجهة اللعب التفاعلي (الـ 7 مراحل)
- الهدف: بناء تجربة اللعب الكاملة مع البطاقات والأسئلة والمراحل
- المخرجات: صفحة اللعب، نظام المراحل، بطاقات الأسئلة والإجابات، نظام النضج

### المرحلة 3: نظام التعارضات + Output Map + تصدير PDF
- الهدف: كشف التعارضات الاستراتيجية وإنشاء خريطة البراند النهائية
- المخرجات: محرك التعارضات، صفحة Output Map، تصدير PDF

### المرحلة 4: وضع الفريق + التحسينات النهائية
- الهدف: تمكين العمل الجماعي والمشاركة وإضافة اللمسات النهائية
- المخرجات: مشاركة Brand Map، وضع الفريق، تحسينات UI/UX
